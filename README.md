# Goo Lab 🧪

> A magical research lab where you raise wobbly slimes through falling-merge puzzles.

魔法薬の研究室で、ぷるぷるスライムを育てる **落下合体 × 混色** パズル。

## 🎮 Concept

スイカゲーム形式の落下合体パズルに **「混色」** という第二軸を加えた新しい遊び。

- 同色同サイズ → **進化**（一段階大きいスライムに）
- 異色同サイズ → **混色**（青 + 黄 = 緑）
- ビーカーから溢れたら GAME OVER

> 「進化させたい欲」 vs 「混ざっちゃう恐怖」のジレンマがゲーム性の軸。

## 🎯 Target Platforms

WebGL (itch.io) → Steam → Mobile → **Nintendo Switch**

## 🛠 Tech Stack

- **Engine**: Unity 6 LTS (URP 2D)
- **Language**: C#
- **Shader**: HLSL (Shader Graph + custom metaball)
- **Physics**: Rigidbody2D / Collider2D
- **Animation**: DOTween
